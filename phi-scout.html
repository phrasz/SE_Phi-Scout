φ  —  Scout

What is φ-Scout?

Space Engineers search contracts give you a center point and a radius — the target is somewhere on or inside that sphere. φ-Scout generates a set of GPS waypoints evenly distributed over that sphere surface so you can systematically sweep every direction without doubling back or leaving large gaps. Paste the generated GPS block directly into the game's GPS panel and fly each point in order.

The Fibonacci Sphere

The waypoints are placed using the Fibonacci Sphere algorithm, which spaces points via the golden angle (137.5°). Because 137.5° is mathematically irrational, no two points ever land in the same place — the result is the most even possible spread across the sphere with zero pole clustering. It's the same pattern sunflowers and pinecones use in nature.

How to Use It

1. Enter the contract center coordinates (paste the GPS string or type X / Y / Z manually).
2. Choose a point count and hit Generate Waypoints.
3. Copy the GPS block and import it into Space Engineers (K key → GPS → Import from clipboard).
4. Fly to each waypoint in sequence — your antenna will ping the target if it falls within detection range of that stop.

Tips

• Orient Y-axis up — align it with planet gravity or your ship's vertical axis and the waypoints form clean latitude rings that are easy to fly in order.
• Reach each waypoint at the stated radius, not inside it — the sphere surface is where the gaps between stops are smallest.
• Start with 12 points. If nothing is found, re-run with 20 or 32 before widening the search area.
• The Lean (9-point) option is the mathematical minimum for a 3 km sphere with 2 km detection (max gap ≈ 2,000 m). It's a fast first pass — if the target isn't detected, step up to 12 or higher.
